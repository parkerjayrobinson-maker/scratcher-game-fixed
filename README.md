# Scratcher Game


🎰 Ultimate 9-Tile Scratcher Deluxe


## Features
- Reveal all tiles when you lose
- Tokens system
- Daily reward (once every 24 hours)
- Saves progress in localStorage (works on refresh/close)
- Shop with brushes


## How to play
1. Create a user
2. Play scratcher game with tokens
3. Claim daily rewards
4. Try to match 3 symbols for bonus tokens!
